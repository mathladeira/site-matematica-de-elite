Política de privacidade

Última atualização: 17/02/2026

  1. Introdução

    Esta Política de Privacidade descreve como o Matemática de Elite coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD).

    Ao utilizar este site ou adquirir nossos produtos e serviços, você concorda com as práticas descritas nesta Política.

  2. Quais dados coletamos

    Podemos coletar os seguintes dados pessoais:

    1. Dados fornecidos pelo usuário:

      Podemos ter acesso a informações como:

      • Nome completo.
      • Email.
      • Número de telefone/WhatsApp.
      • informações enviadas por formulário ou contato direto.

    2. Dados de pagamento:

      Os pagamentos são processados por plataforma terceirzada.

      Não armazenamos dados completos de cartão de crédito.

      Podemos ter acesso a informações como:

      • Nome do comprador.
      • Email.
      • Status do pagamento.

    3. Dados coletados automaticamente:

      Podemos coletar:

      • Endereço IP.
      • Tipo de navegador.
      • Informações de acesso e navegação.
