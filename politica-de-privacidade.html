Política de privacidade

Última atualização: 17/02/2026

  1. Introdução

    Esta Política de Privacidade descreve como o Matemática de Elite coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD).

    Ao utilizar este site ou adquirir nossos produtos e serviços, você concorda com as práticas descritas nesta Política.

  2. Quais dados coletamos

    Podemos coletar os seguintes dados pessoais:

    1. Dados fornecidos pelo usuário:

      Podemos ter acesso a informações como:

      • nome completo;
      • email;
      • número de telefone/WhatsApp;
      • informações enviadas por formulário ou contato direto.

    2. Dados de pagamento:

      Os pagamentos são processados por plataforma terceirzada.

      Não armazenamos dados completos de cartão de crédito.

      Podemos ter acesso a informações como:

      • nome do comprador;
      • email;
      • status do pagamento.

    3. Dados coletados automaticamente:

      Podemos coletar:

      • endereço IP;
      • tipo de navegador;
      • informações de acesso e navegação.

      Esses dados são coletados para fins estatísticos e de segurança.

  3. Finalidade do uso de dados

    Utilizamos os dados pessoais para:

    • processar compras e liberar acesso ao material digital;
    • executar e organizar a mentoria individual;
    • responder solicitações de contato;
    • cumprir obrigações legais;
    • melhorar a experiência no site.

    Não vendemos nem comercializamos dados pessoais.

  4. Compartilhamento de dados

    Os dados podem ser compartilhados apenas quando necessário com:

    • plataforma de pagamento;
    • serviços de hospedagem do site;
    • ferramentas de comunicação;
    • autoridades públicas, quando exigido por lei.

    O compartilhamento ocorre apenas na medida necessária para prestação dos serviços.

  5. Armazenamento e segurança

    Adotamos medidas técnicas e organizacionais razoáveis para proteger os dados contra acesso não autorizado, perda, alteração ou divulgação indevida.

    Os dados são armazenados apenas pelo tempo necessário para cumprimento das finalidades descritas nesta Política ou conforme exigência legal.

  6. Direitos do titular dos dados

    Nos termos da LGPD, você pode solicitar:

    • confirmação da existência de tratamento;
    • acesso aos dados;
    • correção de dados incompletos ou incorretos;
    • exclusão de dados, quando aplicável;
    • revogação do consentimento.

    Para exercer seus direitos, entre em contato por:

    Email: matematicadeelite@gmail.com

    WhatsApp: (91)98543-0575

  7. Cookies

    O site pode utilizar cookies para melhorar a navegação e coletar informações estatísticas.

    O usuário pode configurar seu navegador para bloquear cookies, se desejar.

  8. Alterações nesta Política

    Esta Política de Privacidade poderá ser atualizada periodicamente.

    Recomendamos que o usuário consulte esta página regularmente.
